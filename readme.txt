This plugin helps to remove URL Field from comment form in Astra Theme.

== Description ==
The plugin allows you to disable/Remove the URL/Website field from Comment form in Astra Theme. The Remove URL Field from Comment Form in Astra Theme plugin works in both free version and Premium Version of Astra Theme.

Features:

1. Remove URL Field in Free Version of Astra Comment Box.
2. Remove Website Field from Premium Version of Astra Comment Box.

== Installation ==
#### From within WordPress
1. Visit 'Plugins > Add New'
1. Search 'Remove URL Field from Comment Form in Astra Theme'
1. Install and Activate the plugin.

#### Manually
1. Download and unzip the file.
1. Upload the folder Remove URL Field from Comment Form in Astra Theme plugin into 'wp-content/plugins/' folder.
1. Activate from the plugin page and Enjoy.

The plugins inject the code directly to functions.php to remove URL Field from the comment form in Astra Theme.
